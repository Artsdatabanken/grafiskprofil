Scripts

Her kan du se de våre ulike script

Nye grafiske profiler har ført til endringer og litt nye filer. Vi anbefaler å ikke legge inn nye filer som kalles "new-something" da det ikke fører noe godt med seg. Dersom vi skal legge inn noe nytt som er veldig annerledes og som kan få uante følger på noen av sidene våre anbefaler vi å legge til et versjonsnummer og øke dette fra gang til gang. Ellers bør vi oppdatere nyeste versjonsnummer dersom det er mindre justeringer, slik at tjenestene våre endrer seg i takt med script på denne nettsiden.

Når du legger inn nye filer er det fint om du oppdaterer lista under.
